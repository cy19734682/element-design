## EmCharts

图表组件

### 组件属性

|   参数    |                  说明                  |  类型  | 可选值 | 默认值 |
| :-------: | :------------------------------------: | :----: | :----: | :----: |
|  config   |         配置项（同EChats配置）         | String |   —    | chart  |
| className |               class样式                | String |   —    | chart  |
|    id     | 图表Id(同一页面的不同图标设置不同的Id) | String |   —    | chart  |
|   width   |                图表宽度                | string |   —    | 200px  |
|  height   |                图表高度                | string |   —    | 200px  |
